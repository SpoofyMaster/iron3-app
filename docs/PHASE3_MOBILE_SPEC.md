# Iron3 — Phase 3: Mobile-Ready Feature Spec

## Inspired by Runify screenshots analysis:

### New Features Needed:
1. **Live GPS Workout Tracking** — Map with route, real-time pace/distance/time, rank XP bar during workout
2. **Training Plan System** — Weekly plans with swim/bike/run sessions, planned vs completed tracking
3. **Activity Detail Screen** — GPS map route, time, distance, pace, elevation, cadence, lap splits
4. **Availability Scheduler** — Per-discipline day picker (swim days, bike days, run days)
5. **Share to Instagram** — Shareable workout card with custom background + stats
6. **Friends/Social System** — Follow friends, friends leaderboard with photos + RP
7. **Profile with Social** — Avatar, followers/following, bio, rank badge, weekly stats
8. **Garmin + Apple Watch Integration** — Auto-sync workouts from wearables
9. **Level/XP bar in nav** — Always-visible level indicator + streak in top bar

### Design Patterns from Screenshots:
- Purple/blue/cyan neon glow borders around phone mockups
- Gradient text (blue gradient) for emphasis words
- Circular progress rings for swim/bike/run (not just bars)
- Podium leaderboard with real user photos
- Rank emblems with wings (crystal/diamond style)
- Bottom tab: Home, Friends, + (log), Ranks, Profile
- Level badge (number) + XP bar always in top-left
- Streak fire + count in top-right
